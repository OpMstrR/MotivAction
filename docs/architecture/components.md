Frontend (Mobile/Web App)

Призначення: Інтерфейс для взаємодії користувача з системою;
Основні функції:
 • Авторизація;
• Перегляд квестів;
• Прогрес/XP;
• Смішні покарання;
• Push-нотифікації.
Протокол/інтерфейс: HTTPS REST API;
Дані на:
• Вхід: запити користувача;
• Вихід: UI-відповіді.
Технології: React Native / Flutter / Vue / React.



Backend (Core Logic Service)

Призначення: Бізнес-логіка, обробка квестів, XP, покарань;
Основні функції:
• Генерація квестів;
• CRUD квестів;
• Облік виконань;
• Розрахунок XP;
• Відсилання штрафів.
Протокол: REST API;
Дані CRUD об’єкти: quests, userProgress, punishments
Технології: Node.js / NestJS / Django / Go



Auth Service

Призначення: Реєстрація + Логін
Функції:
• JWT токени;
• Відновлення паролю.
Протокол: /auth/* (REST);
Дані на:
• Вхід: email/password;
• Вихід: токен.
Технології: Firebase Auth.




Database

Призначення: Зберігання даних користувачів, XP, квестів;
Функції:
• Таблиці: Users, Quests, Punishments, XP_History;
Інтерфейси: SQL запити;
Технології: PostgreSQL / MongoDB.




Notifications Service

Призначення: Відсилання пушів та смішних покарань;
Функції:
• Push API;
• Cron перевірка пропущених квестів.
Протокол: WebPush / FCM;
Дані event: missedQuest, newQuest.



Admin Panel

Призначення: Модерація квестів, перегляд статистики;
Функції:
• Додавання квестів;
• Перегляд активності.
Протокол: Web;
Технології: React / Vue.
